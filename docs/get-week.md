# 教务系统获取教学周接口文档

## 接口说明

该接口用于获取当前学期的教学周信息。调用该接口可以获得当前是第几周以及学期的总周数。或“当前日期不在教学周历内”

## 请求说明

请求地址：http://zhjw.qfnu.edu.cn/jsxsd/framework/jsMain_new.jsp?t1=1
请求方法：GET
请求参数：无

## 响应说明

响应格式：HTML
响应内容示例：

非教学周的时候

```html
<!-- 省略 -->
<span class="main_text main_color">当前日期不在教学周历内</span>
<!-- 省略 -->
```

教学周的时候

```html
<!-- 省略 -->
$("#li_showWeek").html("<span class=\"main_text main_color\">第18周</span>/20周");
<!-- 省略 -->
```

## 解析说明

1. 发送GET请求到指定的接口地址。
2. 解析响应内容：
   - 如果响应中包含`当前日期不在教学周历内`，则表示当前不在教学周内。
   - 如果响应中包含类似`第18周</span>/20周`的内容，则可以提取出当前是第几周（如18周）以及学期的总周数（如20周）。
3. 返回解析结果。
